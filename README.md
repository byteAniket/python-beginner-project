# Quiz Game 🎯

This is a simple quiz game built using Python. The game asks multiple questions, takes user input, and calculates the final score based on correct answers.

## Features

- Multiple choice or input-based questions
- Score tracking system
- Simple and interactive gameplay

## Technologies Used

- Python

## How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Open the project folder
4. Run the file:

```
python quiz.py
```
